# Finote

`Finote` — дипломный Android-проект в формате `MVP`.

Приложение предназначено для контроля кредитных карт, задолженностей, лимитов, дат платежей и операций по ним.

---

## Цель проекта

Разработать Android-приложение, которое решает прикладную задачу учёта кредитных карт и одновременно демонстрирует ключевые темы курса Android-разработки:

- `MVVM`
- `Clean Architecture`
- локальную базу данных `Room`
- навигацию между экранами
- `Dependency Injection`
- работу с внешним `API`
- локализацию интерфейса
- современный UI на `Jetpack Compose`

---

## Основные возможности

- отображение списка кредитных карт на главном экране
- отображение долга, лимита, доступного остатка и даты платежа
- добавление карты
- редактирование карты
- просмотр деталей карты
- добавление операции
- редактирование операции
- отображение курса `USD/RUB`
- отображение даты актуальности курса
- отображение источника внешних данных
- локализация интерфейса `RU / EN`

---

## Основные экраны

- `HomeScreen` — список карт и блок курса валют
- `AddCardScreen` — добавление и редактирование карты
- `CardDetailsScreen` — детальная информация по карте и список операций
- `AddTransactionScreen` — добавление и редактирование операции

## Архитектура

Проект построен по схеме:

`UI -> ViewModel -> UseCase -> Repository -> Room / API`

Основные слои:

- `presentation/` — экраны, навигация, `ViewModel`, `UI State`
- `domain/` — модели и `UseCase`
- `data/` — `Room`, `DAO`, `Entity`, `Repository`, `API`
- `di/` — зависимости приложения

---

## Внешний API

Для блока курса валют используется внешний API: [open.er-api.com](https://open.er-api.com/)

## Как запустить проект

1. Открыть проект в `Android Studio`
2. Дождаться `Gradle Sync`
3. Запустить приложение на эмуляторе или Android-устройстве
